Chrome Web Store Source Code

Flexible rules for real productivity

Block time-wasting sites in seconds using flexible rules built for real productivity.

  • Always Block - completely block distracting websites.
  • Daily Time Limits - control how long you can spend on sites.
  • Scheduled Blocking - set recurring focus periods.
  • Tunnel Vision Mode - allow only one site for deep work.

Intentional friction

Unlike typical blockers, WebBlock adds intentional friction when you try to bypass restrictions.

  • Typed confirmations instead of one-click overrides.
  • Built-in 15-minute cooldown before access is restored.

This helps prevent impulsive unblocking and keeps you aligned with your goals.

Ideal for

  • Studying and revision.
  • Deep work sessions.
  • Reducing social media usage.
  • Breaking doom-scrolling habits.

Privacy-first

WebBlock does not collect or transmit your data. All settings are stored locally in your browser.
