What WebBlock does

Simple controls, strict overrides, and fast setup for real, distraction-free focus.

Always Block Instantly block specific websites with one click.
Daily Time Limits Use social apps intentionally without losing track of time.
Scheduled Rules Schedule blocking for work, study, or sleep.
Tunnel Vision Allow one site for a set time while blocking everything else.

Unblock flow with friction

WebBlock requires typed confirmations and a cooldown to prevent impulsive unblocking.

Privacy

WebBlock runs locally in your browser. Settings and timers are stored in Chrome local storage. No account required.
